Computing BSc Information

BSc Computing teaches students various skills and knowledge of modern day computing workplaces. At DMU, we provide you with all the facilities to help you study towards your degree. This is a three year course and each year you will be studying four different modules. Below is a table of all the modules you will work on this year:


Module Code Module Name Credits
IMAT1214 Business Technology 30
IMAT1604 Visual Web Development 30
IMAT1215 Database Management & Reporting 30
IMAT1401 Information Systems Development 30

About the Modules

Business Technology is the module where you will learn technical and creative aspects of computing. You will be taught how to construct a website using HTML and CSS, as well as using popular business software such as Microsoft Word, Excel and PowerPoint.

The next module is Visual Web Development. This module shows you the fundamentals of programming. You will use a program called Microsoft Visual Studio to create a website that can add, edit and delete data. Programming is one of the most trickiest aspects of computing but this module will help greatly.


Database Management and Reporting provides students an understanding of how data is modelled in a management system. You will be designing database by working with Entity-Relationship diagrams and again using Visual Studio to implement these designs.

The last module you will be studying is Information Systems Development. This is quite possibly the most complex module because you will be required to create a working computer system. In this module, you will be shown the many tasks that need to be complete in order to carry out such a project. All of this, may sound challenging but will help you in the future.

How will I learn all of this

For each module, you will have a tutor. They will undertake all of your lab sessions, tutorials and lectures. In lab sessions and tutorials, you will be enhancing your skills and knowledge by completing various learning materials. However, in lectures, the tutors will bring you all together and discuss what work you will be doing over the next couple of weeks, assignment announcements and any other relevant information.


Assessments

Every module will of course have assessments, below is a list of all the types of assessments:

  • Phase Tests
  • Formal Examinations
  • Both Individual and Group Assignments
  • Presentations
  • Research & Reports
